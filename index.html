О сайте

На страницах каталога представлены Блум, Стелла, Флора, Муза, Текна и Лейла. Для каждой героини подготовлены краткое описание, характеристики и подробная информация.

Сайт состоит из главной страницы, каталога и отдельных страниц персонажей. Все страницы объединены единым стилем оформления с помощью внешнего файла CSS. Для меню сайта использованы списки без маркеров, а для списка характеристик — графические маркеры-изображения.
